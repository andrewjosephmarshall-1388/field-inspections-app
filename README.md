# Field Inspections App

A mobile-first inspection app for field workers using MUNIS data.

## Features

- 📱 Mobile-optimized interface
- 📍 Google Maps integration for directions
- 📷 Photo capture and storage
- 📝 Inspection notes
- ✅ Status tracking (Pass/Fail/Pending)
- 📧 Email results
- 📴 Works completely offline
- 💾 Data stored locally on device

## How to Use

1. Open the app on your phone
2. Click "Import CSV" to load your daily inspections
3. Click any inspection to view details
4. Add notes, take photos, mark status
5. Click "Email" to send results
